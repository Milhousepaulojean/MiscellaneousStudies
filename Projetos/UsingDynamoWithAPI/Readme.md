## Rodando localmente

Clone o projeto

```bash
  git clone -b nodejs --single-branch https://github.com/Milhousepaulojean/MiscellaneousStudies.git
```

Entre no diretório do projeto

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
